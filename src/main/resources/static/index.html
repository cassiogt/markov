Conversor CTMC para DTMC

Este programa realiza a leitura de uma matriz CTMC inserida através do campo de texto ou através de arquivo, o converte para DTMC calcula as probabilidades de cada entrada e apresenta teste para comprovação das probabilidades calculadas.

Autores:
Cássio Tatsch (60064)
Igor Dettenborn (75314)

Será gerado número randomico para testar as probabilidades.
